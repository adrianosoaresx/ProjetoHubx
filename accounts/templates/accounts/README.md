# Convenção de templates de confirmação de e-mail

Para evitar ambiguidade entre nomes em PT/EN, este app adota **apenas** a convenção em português para o template de sucesso da confirmação de e-mail:

- `accounts/email_confirm_sucesso.html` (template canônico)

O nome `email_confirm_success.html` foi removido para impedir duplicidade e confusão de referência.
